Cloud Resume Challenge
January 2021

The website you are currently viewing! This website is the end product of Forrest Brazeal's cloud resume challenge. A secure serverless responsive website that utilizes AWS cloud services by employing infrastructure as code, includes a dynamic visitor counter, and has a CI/CD pipeline for easy automated deployment.

  • Created an S3 bucket to store necessary website files to be deployed online as a static website.
  • Purchased and made a custom domain name with Route53, generated a SSL certificate in AWS Certificate Manager, and created a CloudFront distribution to allow the site to be served securely over HTTPS.
  • Implemented the visitor counter using AWS API Gateway, Lambda, and DynamoDB which is then provisioned using CloudFormation.
  • Set up a pipeline in CodePipeline allowing easy automated deployment to the S3 bucket whenever new code is pushed to Git.
